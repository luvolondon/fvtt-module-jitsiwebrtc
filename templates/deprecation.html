Jitsi WebRTC client is deprecated

Deprecation warning

Due to ongoing complications combined with a change from Jitsi that removes access to their public server, this module is now deprecated and no longer maintained.

More information

For more details, please see the github issue: Module is deprecated and unmaintained.

If you are interested in taking over maintenance of the project, please contact bekit#4213 on Discord.

Need a replacement?

The LiveKit AVClient module is available as a more modern alternative A/V Client.
